Introduction

Product Barcode Sticker Generator This add-on allows you to create dynamic barcodes for Active eCommerce products. The “Product Barcode Generator” can generate barcodes based on SKU numbers, and IDs, using a total of 14 formats, including EAN13, UPC (A), and CODE128. Can easily customize barcode width, height, color, and label, from the plugin settings page

Installation Steps

Step 1
Unzip the file you downloaded from Codecanyon, It includes one zip file. Extract the file in your local folder any where. You will get three folders. which contains Views, Controllers, Assets and Routes.


Step 2
Now got to your Active eCommerce CMS application.

Controllers: Go to this directory app\Http\Controllers. In the Controllers folder you need to paste BarcodeController.php which is inside the Controllers folder you just extracted earlier.


Views: Go to this directory resources\views. Now paste the views folder.


Routes: Go to this directory routes. Now paste the barcode.php.


Step 3
We need write few lines of code to some files. Don't panic its so easy even if you are not a tech guy.
First Of all go to app\Providers\RouteServiceProvider.php and find map() function.
$this->mapBarcodeRoutes();
write this to any line. Here is the example:

Then write mapBarcodeRoutes() outside of the map() function.
protected function mapBarcodeRoutes(){
Route::middleware('web')
->namespace($this->namespace)
->group(base_path('routes/barcode.php'));
}


Step 4
Now we need to add Blade directive at AppServiceProvider.php
Let's go the app\Providers\AppServiceProvider.php

You can copy the code from AppServiceProvider.php




Step 5
Now we need to add side menu at resources\views\backend\inc\admin_sidenav.blade.php
just write @barcode_menu under the dashboard or any position that you want


Step 6
Now we need to add styles and scripts at resources\views\backend\layouts\app.blade.php
just write @yield(css) inside head tag.


Now, you can visit the Barcode menu.

Usage Of Barcode

This plugin generates unique barcodes for each product. The barcode will take its content from the product SKU, ID, and automatic barcode number (premium). and the barcode image will be automatically generated based on that.
